台灣地方創生：議題涵蓋、推動策略與「小店永豐計畫」的實踐分析

I. 前言：台灣地方創生背景與核心目標

台灣面臨的挑戰：人口結構與城鄉發展失衡

台灣社會長期以來一直面臨嚴峻的人口結構與城鄉發展失衡問題。總人口數持續減少，同時人口過度集中於都會區，導致城鄉發展差距日益擴大 [1, 2]。這種失衡現象不僅使得地方原有的公共設施，如學校、醫院、郵局和商店等，因人口稀少而逐漸消失 [2]，更造成青壯年勞動力大量外流至都市，僅留下高齡人口在鄉鎮，使得地方傳統文化與技藝因缺乏後繼者而面臨失傳的危機，最終可能導致「地方消失」的困境 [2]。

為了應對這些挑戰，台灣地方創生相關計畫的推動顯示出其新興且充滿活力的特點。根據2023年的調查，超過三分之一（35.1%）的地方創生團隊成立時間在五年內，這反映了近年來各界對此議題的積極投入 [3]。然而，儘管有新血加入，高達70.7%的團隊仍指出「團隊人才不足」是其運營中最大的難題 [3, 4]。這項數據直接呼應了人口外流導致勞動力短缺的深層挑戰，顯示地方創生在人才吸引與培育方面仍需持續努力 [4]。

地方創生的緣起與國家戰略定位

為應對上述嚴峻的社會與經濟挑戰，行政院於2019年1月核定「地方創生國家戰略計畫」，將地方創生提升至國家安全戰略層級的政策高度 [1, 5]。此計畫的核心目標是積極協助地方發掘其獨特的「地方DNA」，透過「由下而上」的方式凝聚在地共識，形成具體的事業提案 [1, 6]。政府在其中扮演關鍵的媒合角色，整合中央部會資源並引導民間投資，以支援地方創生工作的推動，最終旨在實現「均衡臺灣」的國家發展目標 [1, 5, 6, 7, 8, 9]。

地方創生的概念源於日本，但台灣在借鑒其經驗的同時，也發展出符合自身國情的特色。與日本主要以單一行政區域為單位不同，台灣考量到各鄉鎮距離較近的地理特性，更強調將鄰近鄉鎮集結成「生活圈」作為地方創生的推動單位 [2]。這種區域性的整合思維，旨在打破行政區劃的限制，促進更大範圍內的資源共享與協同發展，以期達到更全面的地方活化效果。

地方創生1.0、2.0、3.0的演進與核心精神

台灣的地方創生政策自推動以來，經歷了數個階段的演進，每個階段都反映了對地方發展問題認知的深化與策略的調整。

  • 地方創生1.0 (2019年核定)： 此階段的政策主要聚焦於解決人口外流問題，鼓勵青年返鄉或移居地方 [10]。其策略重心多透過政府補助，鼓勵在地青年創業、發展小型特色產業，或舉辦短期活動以吸引人氣 [10]。當時規劃了五大推動戰略，包括「企業投資故鄉、科技導入、整合部會創生資源、社會參與創生及品牌建立」 [5, 7]。這是一種較為直接且快速見效的策略，旨在應對當時迫切的人口外流問題。
  • 地方創生2.0 (2020年10月核定，推動期程為110年至114年)： 隨著實踐經驗的累積，政府與民間逐漸意識到僅靠「人口回流」不足以支撐地方的長期發展 [10]。為加大推動力道，2.0計畫新增了多元徵案管道、設置在地青年培力工作站、活化公有空間、成立分區輔導中心及專案辦公室等策略，以提高整體資源運用效率 [1, 5, 6, 8, 9]。此階段更強調發展「可獲利並持續自主經營」的商業模式，以及具備在地性、公益性、公共性的「共好」效益，並加強對留鄉/返鄉青年及人才培力的投入 [9]。這表明政府意識到地方創生需要內生動力和永續性，而非僅依賴外部補助。
  • 地方創生3.0 (2024年提出，推動期程為114年至117年)： 為持續作為地方創生青年的堅強後盾，政府續提出「打造永續共好地方創生計畫」，預計再投入60億元 [1, 6, 9]。此階段以「永續、公益、共好」為核心精神，推動五大重點工作：培力在地青年、擴增多元途徑、完善基礎建設、促進跨域合作、深化國際連結 [1, 6, 9]。這不僅是資源的加碼，更是策略層次的提升，旨在建立更具韌性、更具國際視野的地方發展模式。

政策從初期解決「表面問題」（人口外流）轉向解決「深層問題」（缺乏內生經濟動能、社會結構失衡），並從「單點突破」走向「系統性建構」，最終目標是打造一個能夠自我循環、永續發展的生態系統。這也反映了政府在政策設計上不斷學習、調整和優化的過程。

II. 台灣地方創生涵蓋之主要議題與主題

台灣地方創生作為一項國家戰略，其涵蓋的議題遠超單一面向，呈現高度的跨領域整合性，這反映了其作為國家戰略級政策的複雜性和全面性。政策的根本動機是解決人口流失和城鄉失衡，這直接引導了對青年返鄉、就業機會創造的關注。為了留住或吸引人口，必須有可持續的經濟活動，這促使地方發展特色產業、導入科技、建立品牌。僅有經濟不足以吸引人，文化認同、社區連結和社會服務是提升生活品質和歸屬感的關鍵。良好的生活環境和完善的基礎設施是地方發展的必要條件，也是吸引人口和產業的誘因，並體現了永續發展的趨勢。地方創生不是孤立的政策，而是一個「生態系統」的建構過程 [10, 11]。它要求各部會資源的整合 [5, 7, 9, 12]，以及產、官、學、研、社等多方力量的協作 [4, 7]。這種全面性也解釋了為何政策會從1.0的「人口」單點突破，演進到2.0/3.0的「永續共好」生態系思維。

A. 人口與人才活化

地方創生政策的核心目標之一是吸引青年留鄉、返鄉與移居，以解決長期困擾台灣的人口外流和高齡化問題 [1, 2, 4, 5, 6, 7, 8, 9, 10, 13]。為此，國家發展委員會積極設置了「青年培力工作站」，迄今已累計核定106處，旨在建構青年留鄉或返鄉的支援體系，透過實質鼓勵在地蹲點經營地方創生事業經驗者，發揮「母雞帶小雞」的功能，陪伴輔導青年扎根，開創地方創生事業 [1, 6, 13]。此外，自2023年度起，政府也鼓勵有志投入地方創生但經驗尚淺的青年，搭配專業輔導培力課程及獎勵金，將各種具實驗性或創新性的構想付諸行動，2023年度共有63隊獲獎勵 [1, 6]。同時，中興新村地方創生育成村的建立，透過活化閒置空間，設置創生培力及創業基地，累計輔導115組地方團隊進駐 [1, 6]。

在創造在地就業機會方面，地方創生透過發展地方特色產業、引入科技與創新經營手法，為青年提供穩定的在地就業機會，這是吸引人口回流的關鍵驅動力 [2, 4, 5, 7, 8, 9]。例如，屏東霧台魯凱部落發展生態旅遊與文創產品，不僅活絡在地經濟，也提供了就業機會 [14]。新竹湖口「木酢達人」則透過將林業廢木、修樹廢木循環經濟利用，轉化為高附加價值產品，並培訓木工職人，成功翻轉傳統產業困境，吸引有志之士長期投入 [15, 16]。台東延平鄉的「布拉谷」團隊，則善用鄉內主要農作如梅子、鳳梨、苦茶籽，形塑特色品牌，為偏鄉青年創造了「回家」的就業機會 [17]。

地方創生也旨在優化人口結構，減緩城鄉人口過度集中與高齡化趨勢，促進城鄉均衡發展 [1, 2, 4, 5, 6, 8, 13, 18]。南投竹山富州社區便是一個成功案例，該社區透過在地化服務與新產業模式，深化人文關懷與居民連結，榮膺全國首座金質卓越獎，成為永續發展的典範 [16]。這些努力共同構建了吸引和留住人才的生態系統，為地方注入新的活力。

B. 產業創新與經濟發展

地方創生強調發掘地方獨特的「DNA」，並據此發展具在地特色的產業，同時推動產業「六級產業化」，將一級生產（農業、漁業）、二級加工、三級服務（觀光、體驗）整合，以提升產品的附加價值和市場競爭力 [2, 3, 4, 5, 7, 8, 9, 10, 15]。例如，雲林古坑的華山咖啡產業便成功從單純的咖啡種植，發展到結合觀光、餐飲、文化體驗的完整產業鏈，並吸引年輕咖啡師返鄉，透過創新行銷與烘焙技術提升咖啡價值 [10]。台東池上則透過藝術季、有機稻米契作、友善環境耕作等方式，將農業與文化觀光深度結合，吸引藝文人士與遊客，並提升農產品的品牌價值，成為「農藝共生」的典範 [10]。

科技導入與數位轉型是地方產業升級的重要途徑。政策鼓勵運用雲端、大數據、物聯網、行動支付等科技，協助地方產業轉型與升級 [2, 3, 4, 5, 7, 15, 17, 19]。例如，雲林口湖的台灣鯛產業運用科技養殖與系統化管理，不僅提升了生產效率，更將魚皮、魚鱗等廢棄物開發成皮包、膠原蛋白保養品、人工敷料等高附加價值產品，成功將傳統產業鍍金並推向國際市場 [15]。南投的謝江林茶莊也運用科技轉型傳統製茶模式，將茶葉製成美感加值、儲存便利的茶餅 [17]。

觀光與文化經濟的結合是活絡地方經濟的另一重要面向。地方創生鼓勵結合鄉鎮地景人文特色形塑觀光主題，並透過舉辦論壇、展覽市集及國際交流活動，帶動地方消費與能見度 [1, 4, 6, 7, 9, 14, 20]。交通部也積極配合，推動地方創生觀光旅遊環境營造，旨在打造魅力景點、完善優質旅遊環境，追求觀光永續發展 [5, 9, 12]。

在地方品牌建立與市場拓展方面，地方創生透過創新觀點與手法，確認當地的獨特性與核心價值，建立城鎮地方品牌，並將其轉化為創造地方生機的資本 [4, 7, 10, 21]。例如，新竹新埔柿染坊以傳統柿染工藝為基礎，打造了名為「KIRKIR」的品牌，完美融合了地方文化特色與現代設計理念 [10]。雲林西螺的御鼎興手工製醬團隊，則透過「飛雀餐桌行動」，將消費者直接帶到產地，連結農產品行銷與農村議題，逐步在台灣各角落遍地開花 [21]。這些案例顯示，透過品牌化和市場拓展，地方產業得以提升競爭力，吸引更多資源與人潮。

C. 社會福祉與社區共融

地方創生在社會福祉與社區共融方面的推動，雖然在國發會的主要政策文件中不總是直接以「長照」等具體詞彙呈現，但其強調「均衡臺灣」的目標，並涉及優化城鎮機能與偏鄉基礎建設，實質上為長照與社會服務網絡的建構提供了基礎 [1, 5, 6]。在實際案例中，可見到大學生團隊在台南東山協助長期照護服務，顯示民間力量在填補此類需求上的作用 [22]。此外，衛生福利部也參與地方創生計畫，推動「長照社區共生互助」模式 [9]。基督教一粒麥子基金會的長照案例也被視為地方治理的成功模式之一，這說明長照議題是透過跨部會合作和民間參與來實現的 [23]。

社區參與與在地共識凝聚是地方創生「由下而上」推動模式的基石。政策鼓勵產官學研社共同參與，讓各界資金、知識技術及人才投入地方創生事業 [1, 4, 5, 6, 7, 18]。例如，台南東山窯鄉的學生團隊透過長期駐村，與居民建立深厚信任感，提供農產品行銷、策展、文史工作、長期照護、民俗活動協助、生活用品維修等多元服務，這種「被需要」的感覺，讓他們至今仍未曾想過離開這個小鎮，展現了深耕在地的力量 [22]。

地方創生也積極促進跨世代交流與社會連結。例如，宜蘭壯圍的「廢墟餐廳」發起的「牽罟」活動，不僅是創意空間，更承載了永續與文化傳承的夢想，透過此活動促進社區長者與年輕人連結，傳遞土地與文化的溫度 [24]。這說明了地方創生在促進社會凝聚力、彌合世代鴻溝方面的潛力。

D. 文化保存與空間活化

地方創生強調發掘地方獨特的「DNA」，並將其轉化為保存及發揚各地傳統文化與技藝的動力 [2, 8]。文化部積極配合，推動「地方創生文化事業發展計畫」，鼓勵結合有形或無形文化資產推動地方創生，以創造在地就業機會並促進地方人口回流 [9, 12]。成功的案例包括新竹新埔柿染坊，該坊以傳統柿染工藝為基礎，透過設計翻轉，實現了地方文化的復興 [10]。屏東霧台魯凱部落的山林日常生活走讀計畫，則重拾與山林的緊密關係，用傳承下來的智慧打造生態旅遊，並融入魯凱族文化奇幻神話故事，展現了原民文化的活化與傳承 [14]。

閒置空間與老舊建築的再生利用是地方創生的重要環節。國發會補助地方政府辦理公有建築空間的環境整備及活化，提供地方青年或團體實體交流空間，作為地方創生推動及經營場域 [1, 6, 9, 13, 25]。成功的活化案例不勝枚舉，例如高雄鹽埕百貨商場轉型為「高雄銀座聚場」，賦予老社區新風采；台中舊宮原眼科醫院從危樓化身為國際知名景點 [14]。新北山城石碇的「螢火蟲書屋」則利用撿回的廢棄木材改造礦工古厝，使其煥發新生 [14]。此外，雲林虎尾糖廠的獨特五分車景觀也透過活化成為當地觀光亮點 [26]。文化部也設有「私有老建築保存再生計畫」，鼓勵民間參與老屋活化 [25]。

透過設計翻轉，營造具在地特色的歷史文化場域，是促進地方人口回流的有效方式 [5, 7, 27]。例如，桃園大溪的普濟路歷史廊道、南投信義的特色文化街景形塑，都旨在透過改善環境與空間，提升地方的吸引力與文化氛圍 [14]。

E. 環境永續與生態友善

地方創生在環境永續與生態友善方面的努力，體現了對土地的深厚關懷。政策推廣食農教育，旨在連結生活飲食與農業間的關係 [1, 6]，並鼓勵發展友善農業 [3]。在資源循環利用方面，台南官田的「烏金合作社」成功將每年約4千公噸的菱角殼廢棄物轉化為「烏金」，解決了環境污染問題並創造新價值 [17]。豐采菇業則利用廢棄菇包造福瓜果、花卉、草莓等設施栽培農友，降低其成本支出，共同打造環境永續的目標 [19]。台東池上透過有機稻米契作與友善環境耕作，不僅提升農產品品質，也維護了生態環境 [10]。環境部資源循環署也積極推動「資源循環地方創生計畫」，旨在扶植具地方特色之資源循環產業，增加地方相關就業機會，帶動人口回流 [12]。

生態棲地保護與環境整備也是地方創生涵蓋的範疇，例如新北山城石碇的「螢火蟲書屋」不僅活化了空間，也致力於生態棲地保護與復育，透過栽培育蝶植物和吸引螢火蟲幼蟲食物，營造友善的生態環境 [14]。同時，持續強化各地方環境整備，如廢棄老舊空間的活化，是完善基礎建設的重要一環 [1, 6, 9]。

環境、社會和公司治理（ESG）理念的導入與實踐，是地方創生擴大影響力的重要途徑。地方創生鼓勵企業透過ESG資源串接企業採購、供應鏈合作，以擴大地方創生影響力 [1, 4, 6, 11]。例如，資誠永續與國家發展委員會合作舉辦的「企業CSR X 地方創生提案競賽」，成功促成多家企業與地方創生團隊的媒合，將企業資源導入地方發展，實現社會共榮與環境保護的目標 [11]。

F. 基礎建設優化

完善的基礎建設是地方創生發展的基石。地方創生計畫持續強化城鎮機能，特別是偏鄉交通等基礎建設，以完善偏鄉基礎設施功能 [1, 5, 6, 9, 15]。交通部在此方面扮演重要角色，推動地方創生公共運輸服務升級，旨在改善偏鄉居民的交通便利性，提升生活品質 [5, 9, 12]。

公共設施的完善也是地方創生關注的重點。這包括活化公有空間，如將閒置的公有建築空間進行整備與活化，提供地方青年或團體實體交流與經營場域 [1, 5, 6, 9, 13]。此外，觀光溫泉區景觀整備（如苗栗泰安）、歷史廊道及生活展演廣場（如桃園大溪普濟路歷史廊道）、特色文化街景形塑（如南投信義）等，都是提升地方生活品質和吸引力的重要基礎建設項目 [14]。這些基礎建設的優化，為地方產業發展、人口回流和整體生活環境的改善提供了堅實的硬體支撐。

III. 「小店永豐計畫」在地方創生中的角色與實踐

計畫宗旨與目標：支持小店，帶動社會影響力

「小店永豐計畫」是永豐基金會發起的一項重要民間資助計畫，其核心精神在於協助台灣各地具備營業行為的小店能夠長久且豐盈地經營下去 [28]。這項計畫不僅僅著眼於商業經營的成功，更期許這些小店能夠作為串連地方生活的「支點」，為解決社會議題帶來正向的改變 [28]。計畫的具體目標包括將小店發展為社區駐點關懷站、透過創意與核心技能為地方創造更多就業機會，以及推動各種對環境友善的舉措 [28]。這些目標與國家地方創生政策中「永續、公益、共好」的核心精神高度契合 [1, 6, 9]，顯示民間力量在響應國家戰略上的積極作為。

輔導面向與資源提供：資金、專業諮詢與平台串聯

「小店永豐計畫」為獲選的小店提供多元的輔導面向與資源。主要支援包括資金挹注和專業諮詢服務 [28]。此外，計畫更透過建立平台，有效地串聯各界資源，讓更多人看見小店的故事、支持小店的發展，最終賦予地方自主發展的力量 [28]。在計畫的評選標準中，「社會影響力」佔比高達40%，這項高權重指標具體評估小店能否帶動當地經濟或社會福利、鼓舞正向議題，以及串聯夥伴推動共創價值 [28]。這項設計明確地強調了計畫超越單純商業模式的社會公益性。

「小店永豐計畫」的運作模式與政府鼓勵企業社會責任（CSR）和環境、社會、公司治理（ESG）的政策方向相輔相成。國家發展委員會積極鼓勵企業透過ESG資源，串接企業採購、供應鏈合作，以擴大地方創生影響力 [1, 4, 6, 11]。永豐基金會的「小店永豐計畫」正是民間企業響應地方創生理念，透過自身資源投入社會發展的典型案例 [11, 13]，為公私協力推動地方創生提供了具體的實踐範本。

具體案例分析

「小店永豐計畫」已輔導多個具代表性的在地小店，其成功案例清晰地展現了微型經濟體如何結合社會使命，在地方創生中發揮重要作用。

  • 木想傢：偏鄉孩童技藝培力與夢想拓展 (花蓮玉里)
    「木想傢」位於花蓮玉里，其創辦人王嘉納目睹許多偏鄉孩子因缺乏工作機會與學習動力，重蹈父母青壯年離鄉的覆轍 [24]。為扭轉這種命運，他看見孩子們靈巧雙手蘊含的無限潛力，於是開始教授木作技藝，為他們開創全新的未來之路，並由此創立了「木想傢」 [24]。在永豐基金會「小店永豐計畫」的支持下，這些孩子不僅習得一技之長，更有機會站上國際舞台，拓展視野與夢想 [24]。他們的成長不僅成為自身希望的火種，更點亮了整個社區的未來，直接回應了偏鄉就業、青年培力與技藝傳承等地方創生核心議題。
  • 廢墟餐廳：牽罟活動與永續文化傳承 (宜蘭壯圍)
    宜蘭壯圍的「廢墟餐廳」不僅是一個創意空間，更承載了「牽罟」計畫的夢想 [24]。創辦人廖誌汶強調，台灣的土地充滿溫暖，各種創意都能獲得支持。他們計畫以裝置藝術美化沙灘，並進行亞洲最大規模的牽罟活動，強調永續性與文化傳承的結合 [24]。此計畫促進了社區長者與年輕人之間的連結與跨世代溝通，傳遞了土地與文化的溫度 [24]。這說明了地方創生的力量來自於經濟與文化的深度結合，透過以牽罟為主軸的小店計畫，讓壯圍的沙灘成為社會與經濟的全新畫布，為文化保存、跨世代交流與環境永續議題提供了具體實踐。

小店模式對地方創生議題的貢獻

「小店永豐計畫」透過支持微型在地經營者，展現了地方創生「由下而上」的實踐路徑 [1, 4, 5, 6, 7, 18]。這些小店作為地方的「支點」或「毛細血管」，能夠更靈活地回應在地需求，提供客製化服務，並與社區建立緊密連結 [28]。它們不僅創造了直接的在地就業機會，更透過其內化的社會使命與影響力，間接帶動了地方經濟的活絡、文化的傳承與社區的凝聚。這些小店是地方創生生態系中不可或缺的微血管，將資源與活力輸送到最基層的角落，為地方帶來持續的正向改變。

即使是小規模的經濟體，若能結合社會使命與在地特色，也能在地方創生中發揮巨大的槓桿效應，解決宏觀的社會問題。國家發展委員會的政策規劃通常是宏觀的，著重於資源整合、基礎建設和人才培力框架 [1, 9]。然而，「小店永豐計畫」則從微觀層面切入，直接支持在地、具備營業行為的小店。這些小店本身就是地方經濟的毛細血管，與居民生活緊密相連。該計畫的獨特之處在於其對「社會影響力」的強調。例如，「木想傢」不僅是木作坊，更是偏鄉孩童的教育與夢想平台；「廢墟餐廳」不僅是餐飲業，更是文化傳承與跨世代交流的場域 [24]。它們超越了單純的商業功能，內化了社會公益。透過資金、專業諮詢和平台串聯，小店得以突破個體限制，導入外部資源，並與在地夥伴共創價值 [28]。這將單一小店的影響力擴散到整個社區。地方創生的成功不僅依賴政府的大型投資和頂層設計，更需要底層的微型經濟體作為「社會企業」的實踐者，將商業模式與社會議題深度融合。這些小店能夠以更低的成本、更靈活的方式，精準回應地方的痛點（如偏鄉教育、文化斷層、長者陪伴），並透過其日常營運，持續產生正向的社會效益，形成可持續的在地發展模式。這也為企業CSR/ESG的實踐提供了具體且可複製的範本。

IV. 地方創生推動的挑戰與未來展望

當前挑戰

儘管台灣地方創生政策目標宏大且持續演進，但在執行層面仍面臨多重挑戰。

首先，人才不足是地方創生團隊普遍面臨的最大難題。根據調查，高達70.7%的團隊認為「團隊人才不足」嚴重阻礙其發展 [3, 4]。這不僅指勞動力短缺，更包括缺乏具備商業模式建立、數位應用、跨域整合等關鍵能力的專業人才 [4]。這項挑戰直接影響了地方產業的創新與永續經營。

其次，行政流程與跨部門協調的複雜性也為地方創生帶來困擾。地方創生計畫的申請與審核流程常被認為過於複雜且耗時，62.8%的團隊希望政府能簡化流程與溝通 [3]。此外，各部會的資源運用通常有其既定的政策目標與主管權責，導致地方創生事業所需的資源對接耗時且落實不易 [5, 18]。高達43.1%的團隊指出，「跨單位/跨領域的溝通與協調」是政府部門最需改善的問題，其重要性甚至高於資金協助 [3]。中華經濟研究院的研究也指出，儘管國發會嘗試透過「行政院地方創生會報」機制整合資源，但各部會位階與分工明確的特性，使得實質的相互搭配仍具挑戰 [18]。

再者，資金與市場競爭也是地方創生團隊必須面對的現實。雖然資金協助是團隊面臨的問題之一（42.6%） [3]，但更深層的挑戰在於如何建立可長可久的商業模式，提升地方產品的市場競爭力，避免過度依賴政府補助 [4, 9, 18]。部分計畫可能流於短期活動或單打獨鬥，缺乏產業鏈串聯與長期經營策略，難以實現永續發展 [4, 10]。

最後，計畫缺乏長期規劃與自主性也可能影響地方創生的成效。有分析指出，現行政策推動模式可能過於偏重產業經濟和公共建設，相對忽略了人口流動等複雜的社會問題，且可能導致社區發展的自主性受限，權責劃分不清 [18]。這使得地方創生在追求「幸福青鳥」的過程中，可能偏離在地居民的真實需求 [18]。

這些挑戰凸顯了政策設計與在地實際需求之間的「落差」，同時也指明了未來需要搭建的「橋樑」。政府明確將地方創生提升至國家戰略層級，並持續投入大量資源，顯示其解決城鄉問題的決心 [1, 5, 6, 9]。然而，來自基層團隊的回饋卻明確指出人才不足、行政流程繁瑣和跨單位溝通困難是三大痛點，甚至比資金問題更突出 [3, 4]。這表明資源雖有，但其「可及性」和「有效性」受限於執行機制。部會間的「明確分工」而非「相互搭配」導致資源整合不易，且政策可能過於偏重產業經濟和公共建設，忽略了「人」的複雜問題，甚至可能影響社區自主性 [18]。

未來機會

儘管面臨諸多挑戰，台灣地方創生仍擁有廣闊的發展機會。

首先，政府的持續投入與政策優化為地方創生提供了堅實的後盾。行政院持續投入資源，例如地方創生3.0計畫預計再投入60億元 [1, 6, 9]。同時，政府也持續優化政策，如擴增多元徵案管道、強化輔導機制，以提高資源運用效率，並支持不同領域的事業提案 [1, 6, 9]。

其次，企業CSR/ESG合作的興起是地方創生擴大影響力的重要途徑。企業透過環境、社會和公司治理（ESG）理念，將其資源、技術和採購鏈導入地方創生，不僅能實踐企業社會責任，也能為地方帶來新的發展動能 [1, 4, 6, 9, 11, 13]。例如，資誠永續與國發會合作舉辦的「企業CSR X 地方創生提案競賽」成功媒合多家企業與地方團隊，促成實質合作，為地方創生注入商業活水 [11]。

再者，跨域整合與數位轉型將是提升地方創生效益的關鍵。政策鼓勵產官學研社等多方力量進行跨域合作，共同強化團隊的商業模式與發展能量 [1, 4, 6, 7, 9]。同時，善用數位工具和科技導入，不僅能提升地方產業的韌性，也能創造更多元的發展機會 [3, 4, 7, 15]。

最後，深化國際連結為台灣地方創生開拓了新的視野。透過推動國際交流、舉辦國際論壇、進行國際參訪合作，台灣不僅能借鑒他國經驗，也能將自身地方創生的成果推廣至國際，使其成為城鄉多元發展的典範 [1, 6, 7, 9, 20]。

打造永續共好地方創生的願景

台灣地方創生將朝向持續創新和開放合作的方向發展，以應對不斷變化的社會經濟環境，並促進地方經濟和社會的均衡發展 [29]。最終目標是形塑具臺灣特色的永續共好發展模式，讓世界看到臺灣各地的美好與特色產業，達成「均衡臺灣」的願景 [9]。

「小店永豐計畫」和企業CSR合作案例則扮演了重要的橋樑角色。它們能夠以更靈活、更貼近在地的方式，將資源和專業知識導入，彌補政府體系在精準對接在地需求上的不足，並為地方創生注入商業永續性。地方創生政策的成功，不僅取決於中央的頂層設計和資源挹注，更在於如何有效地縮小政策意圖與在地執行之間的「落差」。這需要更深化的「公私協力」模式，讓民間組織和企業不僅是資源的接受者，更是政策的共同設計者和執行者。未來的發展將更強調「在地培力」和「賦權」，讓地方社群擁有更大的自主權和能力去定義和實現屬於自己的「創生」，而非僅是配合政府的單向補助。這也意味著，政府需要從「管理者」轉變為「賦能者」和「平台建構者」，促進多方對話與合作。

V. 結論

台灣的地方創生作為一項國家戰略，其全面性與複雜性日益彰顯。這項策略不僅是為應對人口減少、高齡化及城鄉發展失衡等結構性問題，更是一個涵蓋人口、產業、社會、文化、環境與基礎建設等多面向的綜合性發展藍圖。從「地方創生1.0」聚焦人口回流，到「2.0」強調商業模式與共好效益，再到「3.0」深化永續、公益與國際連結，政策的演進清晰地展現了政府在實踐中不斷學習、調整與深化的過程，最終旨在實現一個更具內生動能、韌性與國際視野的「均衡臺灣」願景。

公私協力與在地賦權在地方創生中扮演著不可或缺的角色。「小店永豐計畫」等民間力量的參與，具體證明了微型在地經濟體在地方創生中的巨大潛力與槓桿效應。這些在地小店能夠以其靈活性與對社區的深度連結，更精準地回應在地需求，將商業模式與社會影響力深度融合。它們彌補了政府體系在細節執行上的可能不足，並為地方創生注入了商業永續性。這也凸顯了未來地方創生能否成功，將高度依賴於政府、企業、在地組織及居民之間更緊密、更具信任基礎的公私協力與在地賦權。

儘管地方創生在推動過程中仍面臨諸如人才不足、行政效率、跨部會協調等結構性挑戰，但政府的持續投入、企業ESG理念的興起、科技導入與國際交流的機會，都為地方創生提供了新的動能。展望未來，應著重於簡化行政流程、強化在地人才的培育與留用、建立更有效的跨域整合平台，並鼓勵地方社群擁有更大的自主權去定義和實現屬於自己的「創生」。透過這些努力，台灣將能逐步實現真正的「永續共好」目標，讓各地的獨特魅力得以綻放，並在全球舞台上展現台灣城鄉多元發展的典範。
