主題一：財報的本質——為什麼要看財報

1.1 一個新創公司的困境：營收增加，卻虧損越大

郭榮芳曾經擔任一家新創公司的顧問。這家新創由幾個年輕且優秀的人創辦，他們非常努力，營收不斷增加。每三個月他們就開一次董監事會，在會上提出很多檢討和改進建議，包括業務策略。

會議中他們提出了損益表，開始討論財務報表。但每次到了財務報表的探討，他們只能得出一個結論：「營收多少，但是我們還是虧損」。到最後，他們只能對彼此說：「革命尚未成功，同志仍需努力」。

這個困境反映了一個根本問題：他們知道營收增加，卻不知道為什麼還在虧損，更不知道應該採取什麼行動。

1.2 財報應該回答的三個關鍵問題

郭榮芳在這次顧問經歷中，提出了三個必須從財報找到答案的問題：

第一個問題：損益平衡點在哪裡？

我們的營業額需要做到多少，才能夠不會虧損？這叫損益平衡點。很多企業經營者不知道這個數字。

第二個問題：為什麼營收越多，虧損卻越大？

這個現象說明什麼？通常代表企業的成本結構有問題——當銷量增加時，某些成本沒有被妥善控制，反而在增加。這需要從損益表的成本與費用項目裡找答案。

第三個問題：募集的資金用到哪裡去了？

這家新創募資可能是為了擴展業務、購置設備、招聘人員。但這些資金投入是否真正幫助了企業創造業績？這些資金能用多久？它是否符合企業未來的投資計畫？

這三個問題都指向同一個根本能力：能否透過財務報表，全盤而快速地掌握公司的現況，並從中找出需要改進的具體問題。

1.3 財報力：分析力與洞察力

張忠謀董事長曾說過：「沒有數量化就不能管理。」這說明了為什麼財務報表對企業經營如此重要——它是唯一能數量化企業運作狀況的工具。

但是，財務報表只呈現現象，不告訴你問題所在。這就是為什麼需要「財報力」。

財報力的定義：解讀企業財務報表，進行分析，並從中掌握企業獲利模式、抽出關鍵因素，進而評估企業可能面臨的風險。這包含兩個能力層次：

分析力：

  • 能讀懂財務報表的數字
  • 能識別報表中最重要的幾個指標
  • 能進行比率分析（不只看金額，還要看比例）
  • 能進行同期比較（與去年同期比、與三年到五年趨勢比、與同業比）

洞察力：

  • 能從這些數字後面，看出企業怎麼獲利
  • 能看出企業的致勝武器是什麼、擅長什麼
  • 能判斷企業對未來是否做了足夠的投資
  • 能識別潛在的經營風險

這兩個能力的最終目的是：為管理和決策提供依據。不管你在工作中做決策，還是在進行個人投資決策，正確的財報觀念都能讓你做出真正獲利的決策。

1.4 為什麼零基礎也必須學會看財報

如果你是公司的經營者或主管，不看懂財務報表意味著：

  • 你不知道公司真實的獲利能力
  • 你看不出資金流向，容易做出錯誤的投資決策
  • 你無法識別危機——很多公司在倒閉前的報表已經出現警訊

如果你是上班族或想要進行投資，不看懂財務報表意味著：

  • 你選不出真正賺錢的公司
  • 你容易被表面的 EPS（每股盈餘）或股價迷惑
  • 你無法識別「地雷股」——那些帳面上賺錢，實際上資金出問題的公司

所以，無論你的職位或身份如何，學會看財報都不是選項，而是必備的基本能力。

主題二：四張財務報表的結構與含義

2.1 為什麼有四張報表，而不是一張

企業的財務狀況複雜，單靠一張報表無法完整呈現。四張報表分別從不同的角度揭露企業的財務真相。

2.2 第一張報表：綜合損益表（損益表）

報表回答的問題：我們公司營收有多少？獲利是多少？

報表結構：損益表通常以「兩年度並列」的方式呈現，讓你能看出企業在不同年度的財務表現。

報表的四個基本要素：

  • 收入：企業銷售產品或提供服務所得的金額
  • 成本：生產產品或提供服務所直接投入的費用（如原料、直接人工、製造費用；或者對買賣業來說，就是進貨成本）
  • 費用：企業為了運作而發生的各種支出，包括推銷費用、管理費用、研發費用等
  • 損益：收入減去成本和費用後的結果

從損益表後面看到的東西：

報表背後有一個非常重要、需要你去洞察的部分：企業到底怎麼樣獲利，也就是獲利的模式是什麼。還有：企業的致勝武器是什麼？擅長什麼？ 對於未來，是否做了足夠的投資以維持或增強這個擅長的領域？

最重要的四個數字（大數法則的第一個原則）：

  • 營業收入
  • 營業毛利
  • 營業利益
  • 本期淨利

這四個數字是損益表的骨幹。掌握它們，損益表就掌握了一半。

2.3 第二張報表：資產負債表

報表回答的問題：企業現在擁有多少資產？這些資產從哪裡來？企業欠了多少錢？股東投了多少錢？

報表結構：資產負債表通常以「某一個特定時點」的形式呈現，比如「110 年 12 月 31 日」。它不是像損益表那樣跨越一整年，而是拍了一個「時間點」的快照。

報表的三個重要因素：

  • 資產：企業投入的資金。要獲利，必須投入資金——這叫「將本求利」
  • 負債：資金的一個來源——從別人那裡來的。包括銀行借款、應付帳款等
  • 權益：資金的另一個來源——企業自己的資金。包括股東投入的股本，以及企業賺到的利潤中分配給股東後剩下的部分

最重要的平衡等式：

資產 = 負債 + 權益

這個等式是資產負債表的本質。資金必然等於其來源（別人的+自己的），這形成了一個永恆的平衡狀態。

從資產負債表後面看到的東西：

  • 企業的資本結構（是靠自有資金還是靠借款在經營）
  • 企業的財務風險程度（負債比例有多高）
  • 企業的資產品質（流動性好不好，有沒有陳舊的存貨）

2.4 第三張報表：現金流量表

報表回答的問題：這一年裡，企業的現金進來了多少？出去了多少？

報表的本質：這張報表是資產負債表和損益表背後的基礎。為什麼？因為損益表呈現的是「會計認列的利潤」，但實際進出的現金可能完全不同。現金流量表就是讓你看到：帳上賺的錢，實際有沒有收到現金。

報表的三個要素：

  • 營業活動的現金流量：企業日常經營活動帶進來或帶出去的現金
  • 投資活動的現金流量：企業購置資產、進行投資帶出去的現金；出售資產帶進來的現金
  • 籌資活動的現金流量：企業向銀行借款或進行現金增資帶進來的現金；發放股利帶出去的現金

2.5 第四張報表：權益變動表

報表回答的問題：股東權益在這一年裡是如何變化的？

報表的地位：這份報表的資料其實已經在其他三個報表裡呈現出來了。所以在實務上，看財務報表時，通常只需要重點關注前三張，而不必逐行分析權益變動表。

2.6 四張報表之間的邏輯關係

這四張報表不是獨立的，而是互相支撐和驗證的：

  • 損益表告訴你企業這一年賺或賠多少
  • 資產負債表告訴你企業用什麼資金去賺（或賠）這些錢
  • 現金流量表告訴你企業賺的這些錢是不是真的收到現金了
  • 權益變動表總結了權益在這一年的所有變化

當你學會同時看這三張（加上必要時的權益變動表），你就掌握了企業財務的全貌。

主題三：損益表——企業怎麼賺錢

3.1 為什麼損益表最重要

在四張報表中，損益表是你最應該優先學會的。為什麼？

因為損益表最直接地回答了一個商業的根本問題：企業怎麼通過銷售賺錢。損益表的邏輯就是商業邏輯的邏輯——銷售減去成本，剩下的就是利潤。

3.2 損益表的核心公式

損益表的所有計算都圍繞著這個基本邏輯：

`

營業收入

  • 營業成本

= 營業毛利

營業毛利

  • 營業費用（推銷費用 + 管理費用 + 研發費用）

= 營業利益

營業利益

+/- 業外收支（利息收入、利息支出、投資收益等）

= 稅前淨利

稅前淨利

  • 所得稅費用

= 本期淨利（也叫稅後淨利）

`

這個公式看起來簡單，但它隱藏了企業經營的所有祕密。

3.3 毛利率：企業的第一道防線

毛利 = 營業收入 - 營業成本

毛利率 = 毛利 / 營業收入

毛利率代表什麼？這是企業的附加價值的第一個體現。

為什麼毛利率這麼重要？因為毛利率高或低，直接反映了企業在市場上有沒有競爭力、有沒有議價能力。

毛利率高的原因有幾種：

  1. 技術領先的優勢：例如台積電、大立光等。台積電的毛利率高達 51.6%，大立光甚至高達 60%。為什麼？因為他們掌握了先進的技術，競爭對手短期內無法複製，所以可以維持高價格和高利潤。
  1. 品牌競爭力：例如 LV、Porsche 等奢侈品品牌。他們的毛利率高，因為消費者願意為品牌支付溢價。上文提到的法式高級料理也屬於這一類——一客 6000-7000 元，背後是品牌、環境、食材的綜合價值。
  1. 市場地位和規模經濟：例如壟斷或寡佔市場的企業，或者通過規模優勢降低成本的企業。

毛利率低的含義：

  • 企業在市場上缺乏特殊競爭力
  • 企業經營的可能是低端、標準化的產品
  • 企業容易陷入價格戰

3.4 營業費用與營業利益

在毛利減去營業費用後，我們得到營業利益。

`

營業毛利 - 營業費用 = 營業利益

`

營業費用包括三大類：

  • 推銷費用：廣告、銷售人員薪資、運送費等，與銷售行為直接相關
  • 管理費用：辦公室租金、行政人員薪資、水電費等，企業運作的基本開銷
  • 研發費用：新產品開發、技術改進、實驗費用等

不同行業的營業費用結構差異很大，這反映了不同企業的商業模式。

例如，台積電的三大營業費用中，研發費用特別高。這代表什麼？代表台積電認為，為了維持技術領先地位，研發投入是不能省的。這個洞察非常重要。

3.5 營業利益率與淨利率

營業利益率 = 營業利益 / 營業收入

淨利率 = 本期淨利 / 營業收入

營業利益率反映的是企業核心業務的獲利能力，不受利息、稅金等非營業因素的影響。

淨利率則是最終的獲利能力，已經扣除了所有的成本、費用和稅金。

為什麼要看兩個利率？

因為它們告訴你不同的故事：

  • 營業利益率高，說明企業的核心業務賺錢
  • 淨利率相對低，可能說明企業要承擔很高的利息（負債太多）或稅負

這個差異本身就是一個信號。

3.6 從台積電和鴻海看兩種不同的獲利模式

郭榮芳用實例詳細分析了台積電和鴻海的損益表（以 110 年度為例）。

台積電的財務數字：

  • 投入的資金：期初 27,607 億 → 期末 37,255 億
  • 資金來源：負債 15,548 億（42%）、權益 21,707 億（58%）
  • 營業收入：15,874 億
  • 營業毛利：8,195 億，毛利率 51.6%
  • 營業利益：6,499 億，營業利益率 40.9%
  • 本期淨利：5,970 億，淨利率 37.6%

鴻海的財務數字：

  • 投入的資金：期初 36,742 億 → 期末 39,088 億
  • 資金來源：負債 23,356 億（59%）、權益 15,732 億（41%）
  • 營業收入：59,941 億
  • 營業毛利：3,621 億，毛利率 6%
  • 營業利益：1,489 億，營業利益率 2.49%
  • 本期淨利：1,538 億，淨利率 2.57%

兩者的根本差異：

台積電：投入資金較大，創造的營收約 1.5 兆，但利潤率極高（淨利率 37.6%）。這代表資本密集、技術密集、高附加價值的商業模式。

鴻海：投入的資金相似，但創造的營收是台積電的 4 倍（約 6 兆），利潤率卻很低（淨利率 2.57%）。這代表依靠經濟規模來賺錢的商業模式。

如何理解這個差異？

台積電的高毛利說明它掌握了獨特的技術，附加價值高。鴻海的低毛利說明它主要從事代工、組裝等標準化業務，附加價值相對低，所以必須靠大規模的製造和銷售來賺利潤。

正因為毛利低，鴻海需要高周轉率——這引向了下一個重要主題。

主題四：資產負債表——企業用什麼資金營運

4.1 資產負債表的核心概念

資產負債表回答了一個根本問題：企業用什麼資金在營運。

要獲利，企業必須投入資金。這個原則叫「將本求利」——有本錢才能賺利潤。

資產負債表告訴你：

  • 企業投入了多少錢（資產）
  • 這些錢從哪裡來（負債和權益）

4.2 資產的構成

資產是企業投入的資金，它們會以各種形式存在：

流動資產（一年內可以轉換成現金的資產）：

  • 現金及現金等價物
  • 應收帳款（還沒收到的客戶款項）
  • 存貨（還沒賣出的產品）
  • 其他流動資產

非流動資產（一年以上才能轉換成現金的資產）：

  • 固定資產（廠房、機器設備、車輛等）
  • 無形資產（專利、商標、營業權等）
  • 長期投資
  • 其他非流動資產

4.3 負債和權益：資金的兩個來源

企業的資金必然來自兩個來源：

負債（別人的錢）：

  • 短期借款
  • 應付帳款（還沒付給供應商的款項）
  • 長期借款
  • 應付債券

權益（自己的錢）：

  • 股本（股東最初投入的資金）
  • 資本公積（股東追加投入的資金，不算股本）
  • 保留盈餘（企業賺到的利潤中，沒有發配給股東的部分）

4.4 最重要的平衡等式及其含義

資產 = 負債 + 權益

這不只是一個會計等式，它揭示了企業資本結構的本質。

負債比例 = 負債 / 資產

權益比例 = 權益 / 資產

這兩個比例告訴你：企業依靠別人的錢和自己的錢各占多少比例。

從台積電和鴻海的例子看：

台積電：負債佔 42%，權益佔 58%。說明台積電傾向於用自有資金營運，對別人的錢（銀行貸款等）的依賴相對較低。

鴻海：負債佔 59%，權益佔 41%。說明鴻海更多地依靠外部融資（銀行借款、應付帳款等），自有資金佔比相對較低。

哪種更好？

這沒有絕對答案。高負債比例意味著：

  • 優點：用更少的自有資金撬動更大的資產，如果經營得當，會提升股東的報酬率
  • 缺點：當景氣不好時，負債還是要還的，企業會面臨更大的風險

低負債比例意味著：

  • 優點：企業抗風險能力更強，在經濟下行時更容易存活
  • 缺點：沒有充分利用槓桿效應，可能降低了資本效率

4.5 應收帳款和應付帳款：營運管理的關鍵

在資產和負債中，有兩項特別重要，直接影響企業的現金流——應收帳款和應付帳款。

應收帳款：客戶欠企業的錢

  • 應收帳款越多，說明銷售了產品但還沒收到錢
  • 如果應收帳款不斷增加，會嚴重影響企業的現金狀況

應付帳款：企業欠供應商的錢

  • 應付帳款越多，說明買了原料但還沒支付
  • 這暫時改善了企業的現金狀況，但最終還是要還的

4.6 存貨：資產還是包袱

存貨在資產中佔有一席之地，但它有一個特殊的性質：

流動性問題：

  • 如果存貨能很快賣出，它就是一個優質資產
  • 如果存貨長期積壓，它就變成了一個包袱——佔據現金，浪費倉儲成本

存貨的隱形成本（郭榮芳特別強調）：

當資產沒有流動，它就會產生成本。這個成本包括：

  • 倉儲費
  • 資金佔用費（這些錢本可以用在其他地方）
  • 陳舊、過時、損壞的風險

所以，資產越多不一定越好，關鍵是資產要能夠流動，要能夠轉換成銷售和利潤。

主題五：現金流量表——賺錢和有現金是兩回事

5.1 最常見的矛盾：帳面賺錢，實際沒現金

這是企業經營中最容易忽視、但也最致命的矛盾。

案例一：一個公司帳面賺了 100 萬，但老闆的帳戶裡根本沒有 100 萬。這是怎麼回事？

通常有以下幾個原因：

  • 應收帳款沒收回來：銷售了 100 萬的產品，但客戶還沒支付
  • 存貨積壓：買進了原料或商品佔用了現金，但沒有被銷售出去
  • 折舊：帳上確認了折舊費用（不是現金支出），但當年支付購置資產的費用（是現金支出）已經在過去年度發生
  • 預付費用：提前支付了租金或保險，費用分年認列，但現金已經流出

這就是為什麼需要現金流量表——它告訴你帳上的利潤，實際上有沒有以現金的形式進入企業。

5.2 現金流量表的三個部分

營業活動的現金流量：企業日常經營產生的現金流入和流出

  • 主要來源：銷售商品或提供服務收到的現金、購買原料或商品支付的現金、支付薪資、支付各種營運費用
  • 重要性：這是企業最重要的現金來源。如果營業活動沒有產生正的現金流，企業終將陷入困境

投資活動的現金流量：企業購置或出售資產時的現金流動

  • 流出：購置固定資產、購買投資品、進行轉投資等
  • 流入：出售資產、處分投資品等
  • 性質：通常是流出，因為企業要成長就需要投資

籌資活動的現金流量：企業向外部籌集資金或向股東分配利潤時的現金流動

  • 流入：銀行借款、現金增資、發行債券
  • 流出：償還貸款、發放現金股利、買回庫藏股

5.3 營業活動現金流量的決定因素

營業活動現金流量往往與損益表的淨利潤有很大的差異。為什麼？

關鍵在於應收帳款、存貨和應付帳款的變化：

應收帳款增加 → 現金減少

當銷售增加但客戶還沒付錢時，帳面利潤增加但現金沒進來。

應收帳款減少 → 現金增加

當企業加快收款或客戶支付時，現金進來。

存貨增加 → 現金減少

當企業買進原料或囤積商品時，現金流出。

存貨減少 → 現金增加

當企業銷售存貨時，現金流入。

應付帳款增加 → 現金增加

當企業延長付款期限時，現金暫時留在企業。

應付帳款減少 → 現金減少

當企業支付供應商時，現金流出。

所以，營業活動現金流量取決於兩個部分：

  1. 營業利潤——企業賺了多少
  2. 營運管理——應收帳款、存貨、應付帳款的管理水準

5.4 折舊和攤銷：不是現金，但影響利潤

在分析現金流量時，有一個重要的調整項：折舊和攤銷。

企業購買一台機器需要花現金，但機器的成本不是一次性在購買年度認列，而是按年份分攤。這個分攤就叫折舊。

例如：購買一台 100 萬的機器，預計使用 10 年，每年折舊 10 萬。

  • 購買年：花了現金 100 萬（現金流出），但損益表上只認列了 10 萬的費用
  • 後續 9 年：沒有現金支出，但每年認列 10 萬的費用

重要概念：EBITDA（稅前利息折舊攤銷前利潤）

`

EBITDA = 本期淨利 + 利息 + 稅金 + 折舊 + 攤銷

`

為什麼要看 EBITDA？因為它代表企業實際產生的「現金利潤」——移除了不涉及現金的折舊和攤銷。

5.5 不同企業生命周期的現金流量特徵

企業在不同階段的現金流量形態不同：

新創事業：

  • 營業活動：通常是負數（還在虧損）
  • 投資活動：負數（需要大量投資建立基礎設施）
  • 籌資活動：正數（需要融資來支撐營業和投資的現金流出）

成長公司：

  • 營業活動：開始轉正（開始賺錢了）
  • 投資活動：持續是負數，而且金額很大（為了支撐快速增長）
  • 籌資活動：可能有借款，但也開始有盈餘能用於還款或分配

成熟公司（如台積電）：

  • 營業活動：穩定的正數（持續賺錢，而且現金能收回來）
  • 投資活動：適度的負數（持續進行必要的投資維持競爭力）
  • 籌資活動：通常是負數（用賺到的錢發放股利，逐漸降低負債）

衰退公司：

  • 營業活動：轉為負數（開始虧損）
  • 投資活動：減少（不再積極投資）
  • 籌資活動：正數（可能需要借款來彌補虧損）

5.6 健康永續的現金流量是什麼樣的

郭榮芳歸納了幾個關鍵特徵：

第一個特徵：營業活動產生的現金流量是正數，而且長期穩定

這意味著企業的商業模式在真實地產生現金，不是靠融資或出售資產在維持。

第二個特徵：營業活動產生的現金能覆蓋一般性的投資活動

企業用自己賺的錢來進行日常的資本支出（購置設備、維護基礎設施等），而不是依靠借款。

第三個特徵：覆蓋投資後還有剩餘，用於發放現金股利

這個剩餘就叫「自由現金流量」（Free Cash Flow）。

`

自由現金流量 = 營業活動現金流量 - 資本支出 - 稅金

`

自由現金流量代表企業可以自由應用的現金，可以用來發股利、擴張、還債或預留現金應對不景氣。

5.7 台積電的現金流量為什麼被日本媒體稱為「藝術品」

郭榮芳詳細分析了台積電 105-110 年的現金流量表，揭示了台積電的資金規劃為什麼如此完美：

台積電的特徵：

  1. 營業活動現金流量長期穩定為正：

- 105 年：6,000+ 億

- 108 年：9,000+ 億

- 109 年：9,000+ 億

- 110 年：11,121 億

  1. 投資活動持續且適度：

- 每年進行大量資本支出（購置最先進的製造設備）

- 105-109 年：3,000-5,000 億

- 110 年：8,363 億（因應需求增加）

  1. 營業現金流能完全覆蓋投資並有剩餘：

- 110 年營業現金流 11,121 億 > 投資現金流 8,363 億

- 剩餘用於發放現金股利：2,657 億

  1. 自由現金流量長期穩定成長：

- 整個 6 年期間（105-110 年），自由現金流量呈現穩定增長趨勢

- 發放的現金股利也逐年增加（從 3 元、5 元、8 元、提升到 10+ 元）

為什麼這被稱為「藝術品」？

因為這代表了一個完美的資金良循環：

  • 用利潤支應投資（不是靠借款）
  • 投資創造未來的競爭力
  • 競爭力帶來更高的利潤
  • 利潤既用於再投資，也回饋給股東

這正是郭榮芳想強調的：真正優秀的企業，不只是賺錢，更重要的是有健康、永續的現金流量。

主題六：如何看懂財報——大數法則與比率分析

6.1 財報看起來很複雜，如何入門

很多人看到財務報表時的第一反應是：太複雜了，我怎麼進入？

郭榮芳的答案是：掌握一個原則——大數法則，複雜性就會大幅降低。

大數法則有三個層次：

第一層：從最大的數字抓起來，然後分層分類

資產負債表的最大數字就三個：資產、負債、權益。掌握這三個，資產負債表就掌握了一半。

損益表最重要的數字有四個：營業收入、營業毛利、營業利益、本期淨利。掌握這四個，損益表的主體就清楚了。

第二層：不只看金額，一定要看比率

毛利率 = 毛利 / 營業收入

營業利益率 = 營業利益 / 營業收入

淨利率 = 本期淨利 / 營業收入

負債比率 = 負債 / 資產

權益比率 = 權益 / 資產

為什麼要看比率？因為比率移除了規模的影響，讓你能夠比較不同大小的企業。

例如，台積電的淨利 5,970 億看起來遠大於一家小公司的淨利 10 億。但計算淨利率後，台積電是 37.6%，小公司可能是 15%，比率的對比就一目了然了。

第三層：做比較，不是看絕對值

  • 與去年同期比：看企業的成長趨勢
  • 看三年到五年的資料分析：看企業的中期表現
  • 與同業比較：看企業在行業中的位置

單一年份的數字沒有太多意義。一個企業的淨利率從 10% 下降到 8%，如果是行業整體下滑（所有企業都下滑了），那可能還算正常；但如果同業的淨利率還維持在 10%，那就說明這個企業出現了問題。

6.2 用大數法則分析台積電和鴻海：實戰例子

郭榮芳用台積電和鴻海的數據示範了如何應用大數法則。

第一步：抓出最大的四個數字

台積電（110 年）：

  • 營業收入：15,874 億
  • 營業毛利：8,195 億
  • 營業利益：6,499 億
  • 本期淨利：5,970 億

鴻海（110 年）：

  • 營業收入：59,941 億
  • 營業毛利：3,621 億
  • 營業利益：1,489 億
  • 本期淨利：1,538 億

第二步：計算比率

台積電：

  • 毛利率：8,195 / 15,874 = 51.6%
  • 營業利益率：6,499 / 15,874 = 40.9%
  • 淨利率：5,970 / 15,874 = 37.6%

鴻海：

  • 毛利率：3,621 / 59,941 = 6%
  • 營業利益率：1,489 / 59,941 = 2.49%
  • 淨利率：1,538 / 59,941 = 2.57%

第三步：分析對比

現在清楚了：

台積電：毛利率 50%+，營業利益率 40%+，淨利率 37%+。這些比率都非常高，說明台積電掌握了高價值的產品和市場。

鴻海：毛利率只有 6%，營業利益率 2.5%，淨利率 2.6%。所有比率都很低，說明鴻海依靠的是什麼？

6.3 為什麼資產周轉率對鴻海這樣的公司至關重要

這引入了下一個重要概念。

主題七：企業獲利的兩種模式——高利潤 vs. 高周轉

7.1 法式料理 vs. 餃子店的啟示

郭榮芳用一個貼近生活的例子開始這個單元：

一位客戶很羨慕一家大直的法式高級料理。一客 6,000-7,000 元，食材精緻，空間高雅，而且經常客滿，需要三個月前才訂得到。客戶說：「做生意就要像這樣子，單價很高，利潤率一定很高。」

但另一方面，他自己的公司很多產品線利潤率都很低，他為此很沮喪，不知道怎麼能賺錢。

郭榮芳問他：「你覺得法式料理比較賺錢，還是像八方雲集這樣的餃子店比較賺錢？」

客戶毫不思索地說：「當然是法式料理，單價這麼高，一定賺很多錢。八方雲集賺的都是小錢。」

但這個判斷是錯的。 而差別在於一個被大多數人忽視的因素：速度。

7.2 翻桌率與周轉率的魔法

在餐廳業，有個專業術語叫「翻桌率」——一天內這個位置被客人用過幾次。

法式料理一天能翻幾桌？最多 3 桌。為什麼？因為一頓法式料理要吃 2-3 個小時。

餃子店一天能翻幾桌？40-50 桌。而且很容易外帶，流動性更強。

這就是速度的魔法：

`

低利潤 × 高速度 = 高利潤

`

數學很簡單：

  • 法式料理：6,000 元 × 3 桌 = 18,000 元/天
  • 假設毛利 60%，毛利 = 10,800 元/天
  • 餃子店：200-300 元 × 50 桌 = 10,000-15,000 元/天
  • 假設毛利 30%，毛利 = 3,000-4,500 元/天

看起來法式料理贏了。但我們還要考慮投資成本和日常開銷：

  • 法式料理：裝潢豪華（幾百萬起跳）、租金貴（月租可能 30 萬+）
  • 餃子店：裝潢簡樸（幾十萬）、租金相對便宜（月租可能 5-10 萬）

如果考慮每天的淨利潤和投資報酬率，八方雲集這樣的高周轉模式，在某些指標上可能完全不輸法式料理。而且，一旦開多家分店，規模優勢會更明顯。

郭榮芳強調：速度會創造利潤。

7.3 資產周轉率：低利潤公司的致勝武器

現在引入一個關鍵的財務指標：

`

資產周轉率 = 營業收入 / 平均總資產

`

其中平均總資產 = （期初總資產 + 期末總資產）/ 2

資產周轉率代表什麼？你投入的資產能創造多少營收。比率越高，說明資產利用效率越高。

郭榮芳列出三家公司的數據（109-110 年）：

台積電：周轉率 0.49

鴻海：周轉率 1.53-1.58（是台積電的 3 倍）

八方雲集：周轉率 1.27-1.40（也是台積電的 3 倍）

這個差異背後是什麼？

台積電的模式：投入很多資產（晶圓廠需要大量資本投入），但每單位資產只能產生有限的營收。為什麼還能獲利？因為利潤率極高——每元營收創造 37.6 分的淨利。

鴻海和八方雲集的模式：投入較少的資產（相對而言），但通過快速流轉，每單位資產能產生更多的營收。利潤率低，但速度高彌補了這一點。

7.4 資產報酬率公式：連接利潤率和周轉率

最終，企業的獲利能力可以用一個綜合指標來衡量：

`

資產報酬率（ROA）= 本期淨利 / 平均總資產

`

這個公式也可以拆解為：

`

ROA = （本期淨利 / 營業收入）× （營業收入 / 平均總資產）

= 淨利率 × 資產周轉率

`

這個公式揭示了一個根本的商業真理：

企業獲利有兩個途徑：

  1. 提高淨利率（讓每元營收產生更多利潤）
  2. 提高資產周轉率（讓資產快速流轉）

最理想的狀況是既高利潤又高周轉，但現實中幾乎不存在。通常企業只能擇一：

高淨利率的企業（如台積電）：

  • 依靠技術、品牌或市場地位
  • 周轉率必然較低（大量資本投入換取高品質、高價值）
  • 如果利潤率下降，企業就陷入危機

高周轉率的企業（如鴻海、八方雲集）：

  • 依靠規模、效率、低成本
  • 利潤率必然較低
  • 必須時刻關注成本，不能讓成本失控

7.5 高利潤率的企業需要持續投資

這引出了另一個重要洞察。

台積電的毛利率 51.6% 是憑空掉下來的嗎？不是。

看台積電的費用結構：

  • 推銷費用：相對較低
  • 管理費用：適度
  • 研發費用：非常高

為什麼台積電敢在研發上投入這麼多錢？因為它知道，只有持續的技術創新，才能維持高利潤率。今年不投資研發，明年就可能被競爭對手超越，利潤率就會下降。

相反，低周轉率的企業最害怕什麼？被迫降價進行價格戰。一旦利潤率被壓低，由於周轉率本來就低，就沒有任何緩衝空間了。

所以，郭榮芳給出的建議是：

如果你是高利潤率企業：要讓你的利潤率發揮到極致，你一定要投資在附加價值上——技術、品牌、服務品質。

如果你是低利潤率企業：要讓高周轉率創造利潤，你要投資在運營效率和規模上——供應鏈優化、流程自動化、擴大市場佔有率。

這不是兩種模式的高下之分，而是根據自己的商業模式，把自己的優勢發揮到極致。

7.6 個人生涯的啟示

郭榮芳進一步延伸了這個觀點到個人生涯規劃：

高利潤率公司有點像專業型人才：

  • 在某個領域不斷深化專業
  • 因為專業稀缺，所以議價能力強（高利潤）
  • 但受限於單人的時間和能力（周轉率低）
  • 需要持續投資（進修、研究）來維持優勢

高周轉率公司有點像通才型人才：

  • 跨越多個領域，綜合能力強
  • 時間和精力能應用到更多項目（周轉率高）
  • 單項的付費相對較低（利潤率低）
  • 需要不斷拓展，維持總體收入

郭榮芳的觀點是：了解自己屬於哪種型態，然後就在那個方向上發揮到極致，就會成功。

主題八：不能省的成本——關鍵要素投資

8.1 景氣不好時為什麼有些成本千萬不能省

當景氣不好時，企業都想節約成本。但郭榮芳提出了一個反直覺的觀點：有些成本，越是景氣不好越不能省。

因為某些成本，它們直接決定了企業的競爭力和收入。省了這些成本，短期省了錢，長期卻會失去收入和利潤。

8.2 毛利率背後隱藏的投資

以台積電為例。台積電的毛利率為什麼這麼高（51.6%）？

答案不是因為台積電的產品定價高，而是因為台積電掌握了獨特的技術，使得它能以較低的成本生產高價值的產品。

但這個技術優勢是怎麼來的？

看台積電的營業費用結構：

  • 推銷費用：佔毛利的比例相對較小
  • 管理費用：佔毛利的比例適度
  • 研發費用：佔毛利的比例非常高

如果台積電在景氣差時減少研發投資，短期內毛利率可能不會下降（已經有的技術還在帶來利潤）。但長期呢？競爭對手會不斷進步，台積電的技術優勢會逐漸被蠶食，最後毛利率就會下降。

這就是為什麼台積電即使在景氣周期中，也持續保持高額的研發投入。

8.3 附加價值的三個來源

毛利率高代表企業有附加價值。這個附加價值從何而來？

第一個來源：技術領先

如台積電、大立光。他們掌握了同行難以複製的製造技術，因此能維持高價格。

第二個來源：品牌競爭力

如 LV、Porsche、法式高級料理。消費者願意為品牌支付溢價。

第三個來源：市場地位

如壟斷或寡佔市場的企業，或擁有獨特資源的企業。

但是，無論哪種來源，都需要持續的投資來維持。

  • 技術優勢需要研發投資
  • 品牌優勢需要品牌建設投資（廣告、維持品質等）
  • 市場地位需要投資來維持競爭力

8.4 「資產沒有流動就會產生成本」的深層含義

郭榮芳強調了一個看似反常但極其重要的概念：資產沒有流動，反而會產生成本。

例如：

  • 囤積的存貨需要倉儲費用，佔用的資金可能有機會成本，還面臨陳舊、損壞的風險
  • 應收帳款遲遲無法收回，企業的資金被鎖定，無法用於其他用途

所以，即使在景氣差時，也要維持良好的資產流動性。這不是成本，而是投資——投資於企業的健康。

8.5 關鍵成本與非關鍵成本

如何區分什麼是「不能省的成本」，什麼是可以省的成本？

不能省的成本（關鍵要素投資）：

  • 直接影響企業的核心競爭力的成本（研發、品牌建設、員工培訓等）
  • 維持企業正常運作的必要成本（設備維護、基礎設施等）
  • 關係到資產流動性的成本（應收帳款管理、存貨管理等）

可以省的成本（非關鍵的行政支出）：

  • 不直接涉及核心業務的開銷（某些行政費用、非必要的出差等）
  • 可以通過流程優化而減少的成本（重複性的工作、浪費性的開支等）

8.6 錯誤的應對方式

有些企業在景氣差時犯了一個常見的錯誤：盲目削減成本，包括關鍵要素投資。

例如：

  • 停止研發項目以節省支出
  • 減少員工培訓預算
  • 推遲重要的設備更新

短期內，財報看起來改善了（費用減少了）。但長期的結果是公司的競爭力被逐步削弱。等到景氣回暖，公司發現自己已經落後於競爭對手，很難追趕。

郭榮芳的建議是：在景氣不好時，聰明的公司會保護其關鍵要素投資，而是在非關鍵的、浪費性的支出上進行削減。

這需要管理層對自己的業務有深刻的理解——知道什麼是關鍵，什麼是浪費。

主題九：資產流動與營運管理

9.1 為什麼應收帳款、存貨、應付帳款這麼重要

在資產負債表和現金流量表中，有三個項目反覆出現，對企業的現金狀況影響最大：

應收帳款（應該收到的款項）

存貨（還沒賣出的產品）

應付帳款（應該支付的款項）

這三項的管理水準，直接決定了企業的現金流量。

9.2 應收帳款的危害

如果應收帳款不斷增加，看起來銷售增加了（營收增加），但實際上企業的現金沒有進來。

這會產生什麼後果？

第一個後果：現金壓力

企業賺的錢沒有收到，但該支付的費用（員工工資、原料款項等）還是要支付。長期下去，企業就會陷入現金短缺的困境。

第二個後果：逾期無法收回的風險

應收帳款拖得越久，收不回的風險越高。企業最終可能被迫提列呆帳，直接衝擊利潤。

9.3 存貨的管理

存貨如果管理不好，有兩個極端：

存貨過多：

  • 佔用大量現金
  • 倉儲成本高
  • 面臨陳舊、損壞的風險
  • 市場變化時可能無法銷售

存貨過少：

  • 可能無法滿足市場需求，失去銷售機會
  • 在緊急訂單時無法應對

郭榮芳強調了一個案例中的異常現象：一個公司的存貨售貨天數異常低（3.39 天就賣出去了）。

表面上看，存貨流轉得很快，這是好事。但如果與該公司的銷售增長模式一起看，有可能暗示了什麼？可能暗示了銷售數字中有造假的成分——把沒有真實銷售的產品計入了銷售，所以帳面上顯示的是「非常快速的流轉」。

這提醒我們：異常好的數據，有時也是一個警訊。

9.4 應付帳款的角色

應付帳款是企業向供應商借用的「免息貸款」。延長付款期限，企業可以暫時保留現金。

但是，應付帳款不能無限延長。最終還是要付的。如果企業系統性地延長付款期限，供應商會：

  • 要求更高的價格以彌補資金成本
  • 拒絕繼續供貨
  • 要求現金預付

所以，應付帳款雖然能改善短期現金狀況，但過度依賴會傷害企業與供應商的關係，長期反而增加了成本。

9.5 營運管理的關鍵指標

應收帳款收現天數：

平均多少天才能收到客戶的款項？

`

應收帳款收現天數 = 應收帳款 / 日均銷售額

`

收現天數越短越好，說明企業的銷售收款能力強。

存貨平均售貨天數：

存貨平均需要多久才能銷售出去？

`

存貨售貨天數 = 存貨 / 日均銷售成本

`

售貨天數越短越好，但不能異常短（可能暗示問題）。

應付帳款支付天數：

企業平均多少天才支付供應商款項？

`

應付帳款支付天數 = 應付帳款 / 日均購置成本

`

適度延長支付期限對現金流有幫助，但要平衡與供應商的關係。

營業周期：

`

營業周期 = 應收帳款收現天數 + 存貨售貨天數 - 應付帳款支付天數

`

營業周期代表從支付原料款到收回銷售款需要多久。周期越短越好，因為企業需要佔用的資金越少。

9.6 營運管理如何影響現金流量

郭榮芳用一個具體案例展示：

一家企業的應收帳款收現天數，從 135 天增加到 205 天。這意味著什麼？

意味著企業銷售的錢，收回來用了越來越長的時間。帳面上說賺了錢，但現金一直沒進來。

結合投資活動（企業還在進行大量投資），這家企業的資金壓力會越來越大。最終，當應收帳款無法全部收回時，企業被迫提列呆帳，損益表才反映出真實情況——大幅虧損。

主題十：明明賺錢為什麼還會倒閉——黑字倒閉的警訊

10.1 黑字倒閉是什麼

「黑字倒閉」指的是企業的帳面上還在賺錢（損益表顯示利潤為正），但企業卻因為現金不足而倒閉。

這聽起來很矛盾，但實際上非常常見。郭榮芳提供了幾個具體案例。

10.2 案例一：製造機殼的公司

財務報表表現：

  • 94 年：營收 84 億，淨利 4.2 億
  • 95 年：營收 73 億，淨利 1 億多
  • 96 年：營收 69 億，淨利虧損 11 億
  • 97 年 8 月：資金不足，跳票

表面分析：

  • 94-95 年帳面上賺錢
  • 流動比率 2.5（流動資產是流動負債的 2.5 倍）——看起來很安全
  • 負債比率 43%——沒有超過 50%，看起來還好

但實際發生了什麼：

看現金流量：

  • 93 年：帳面淨利 5.9 億，加上折舊攤銷，EBITDA 7.4 億，但營業活動現金流是 -11 億（現金流出）
  • 94 年：帳面淨利 4.2 億，EBITDA 6.1 億，但營業活動現金流是 -7.8 億（現金流出）
  • 95 年：帳面淨利 1 億，營業活動現金流是 4.3 億（終於有正數）

三年合計：

帳面淨利總共賺了 11 億多，但營業活動現金流是 -15 億（淨流出）。

原因在哪裡：

應收帳款不斷增加：

  • 93 年：30 億
  • 94 年：40 億
  • 95 年：42 億
  • 96 年：49 億

企業的錢都卡在應收帳款裡。帳面上說賺了錢，但實際上客戶還沒付款。

到了 96 年，應收帳款的問題變得不可迴避，企業被迫提列呆帳（認為無法收回的部分），導致 96 年大幅虧損 11 億。

結論：這家企業的真實情況是：產品銷出去了，但客戶沒有按時支付（或根本無法支付）。企業需要支付員工工資、原料款項，但現金沒進來。最終在 97 年資金耗盡，跳票。

10.3 案例二：太陽能公司

財務報表表現：

  • 2006 年：營收 33.9 億，EPS 12 元
  • 2007 年：營收 59.7 億，EPS 12.64 元
  • 2008 年：營收 135.9 億，EPS 8.83 元

看起來營收在爆發式增長，EPS 也不低。

但隱藏的問題：

  1. 營收成長伴隨利潤率下降：

- 毛利率：25% → 13% → 12%

- 這不是「價值型成長」，而是「利潤率被壓縮的成長」

  1. 現金流量的差距很大：

帳面上賺了 27 億左右（3 年合計），但營業活動現金流是負的或非常小。

  1. 融資和投資的惡性循環：

- 營業活動沒有現金流入，企業卻在進行大量投資（購置生產設備以應對訂單增加）

- 投資活動 2008 年就高達 60 億

- 錢從哪裡來？從融資活動：借款每年增加 30-50 億

  1. 資本支出的爆炸：

- 3 年的資本支出總計 58 億

- 無形資產增加 54 億

- 總計 112 億的資本支出，但營業現金流無法支應

真實情況：

企業在太陽能產業蓬勃發展時（2006-2008），接到大量訂單，營收爆發。但為了滿足這些訂單，需要大量的設備投資和前期預付（預付貨款給供應商以搶訂單）。

結果：

  • 帳面上說賺了很多錢（高銷售額）
  • 但現金沒有進來（供應商要求預付、客戶延遲支付）
  • 企業只能靠借款支撐
  • 到了 2009 年，太陽能市場崩潰（中國補貼政策改變，太陽能價格大幅下跌）
  • 企業無法消化之前的高投資，資金危機爆發

這家公司最後被迫進行巨額增資（英業達以 22 元／股增資），被外資接手。

10.4 黑字倒閉的四個常見原因

根據這些案例，郭榮芳歸納了黑字倒閉的四個常見原因：

1. 應收帳款增加，但沒有收回現金

企業銷售增加，看起來營收增加、利潤增加。但實際上，客戶的款項沒有按時支付或無法支付，導致企業的現金始終短缺。

2. 存貨積壓

企業購進了原料或囤積了商品，佔用了現金。如果存貨無法迅速銷售，現金就被鎖定了。

3. 營收成長伴隨利潤率下降

企業為了維持增長，不斷降價或進行大額投資。結果營收增加了，但利潤率卻下降了。投資需要的現金支出，遠大於盈利帶來的現金流入。

4. 過度投資，營業現金流無法支應

企業為了未來的增長而進行大量投資（固定資產、研發等）。但投資需要現金，而營業活動的現金流卻無法覆蓋這些投資。企業最終依靠融資（借款、增資）來支持投資。一旦融資渠道關閉（銀行不願再貸款，市場對公司前景喪失信心），企業就陷入困境。

10.5 黑字倒閉背後的管理失誤

郭榮芳指出，黑字倒閉往往不是因為業務不好，而是因為管理層沒有很好地理解現金流量與利潤的差異。

許多管理層看著損益表說「我們賺錢了」，就覺得企業運作正常。但他們忽略了現金流量表，沒有看到「我們賺的錢沒有收回現金」。

這就是為什麼郭榮芳一開始強調：掌握公司狀況，一定要從財務報表著手，而且要同時看損益表和現金流量表，絕對不能只看其中一張。

主題十一：賺錢企業 vs. 地雷股——風險評估的檢查清單

11.1 什麼是地雷股

「地雷股」是指那些帳面上看起來還不錯（甚至在賺錢），但實際上隱藏著重大財務風險的股票。

這些公司最終可能面臨資金危機、被迫大幅減值，或直接倒閉。

11.2 地雷股的常見徵兆

郭榮芳基於多年的諮詢經驗，列出了幾個地雷股的常見徵兆：

徵兆一：應收帳款顯著偏高或顯著高於同業

這意味著：

  • 企業的銷售款項沒有有效收回
  • 企業的銷售信譽控制有問題
  • 有可能銷售中存在虛構成分

徵兆二：與此同時，存貨顯著偏低

結合徵兆一，這會產生一個可疑的組合：

  • 銷售很多（導致應收帳款高）
  • 但存貨很少（已經全部銷售出去了）
  • 這種組合在現實中是罕見的（通常高銷售會伴隨一定的存貨），有可能暗示銷售中有造假

徵兆三：營業活動現金流量顯著低於獲利，或經常為負數

這是最直接的危訊號：

  • 帳面說賺了錢
  • 但實際上沒有現金流入
  • 或者說，賺的錢沒有收回來

徵兆四：投資不斷出去，營業活動沒有現金進來，錢從融資活動來

這形成了一個危險的循環：

  • 企業在進行大量投資（可能是為了維持成長，也可能是為了支撐虛假的業績）
  • 營業活動產生不了現金
  • 企業只能靠借款或增資來支持投資和日常運作
  • 這種模式是不可持續的，最終會因為融資枯竭而崩潰

徵兆五：負債比例顯著偏高

這加重了風險：

  • 企業已經高度依靠外部融資
  • 一旦現金流出現問題，償還能力會迅速下降

11.3 徵兆的組合判斷法

郭榮芳強調，單個徵兆不足以確定一家公司是地雷股，但徵兆的組合很重要。

  • 出現 1 個徵兆：需要注意，但不必恐慌
  • 出現 2 個徵兆：要進一步分析，警惕度提升
  • 出現 3 個徵兆：危險信號閃起，慎重對待
  • 出現 4 個或以上徵兆：很可能就是地雷股，要特別謹慎

11.4 其他可疑的信號

除了上述徵兆，還有其他需要注意的情況：

徵兆六：過度的業外投資或投資效能顯著偏低

有些企業帳面上的利潤，主要來自於業外投資（房地產投資、股票投資等），而不是核心業務。

這有幾個問題：

  • 業外投資通常不可持續（不像核心業務有持續的客戶和收入）
  • 如果投資效能偏低（投入很多但回報不足），說明管理層的投資決策有問題

徵兆七：銷售客戶或應收帳款集中在少數客戶

危險在於：

  • 企業過度依賴少數客戶
  • 如果這些客戶失去，營收會大幅下滑
  • 如果這些大客戶的規模或資本額與銷售金額不相當，有可能是「實質關係人」（如相關方、關聯企業），甚至是虛構的

徵兆八：公司或集團的組織架構非常複雜且沒有必要

這可能暗示：

  • 企業在進行不透明的資金流動
  • 企業的控制結構不清楚，股東權益可能被稀釋
  • 歷史上的反面教材（如力霸集團）就是因為組織架構異常複雜，最終被查出資金挪用和詐欺

11.5 虛假報表的特徵

郭榮芳進一步指出，當企業報表出現上述組合徵兆時，有可能不只是經營困難，而是報表本身存在虛假成分。

常見的虛假操作包括：

虛假銷售：

  • 企業與關聯方進行循環銷售（賣給它，它再賣給別人）
  • 企業向無力支付的客戶銷售，帳面認列銷售但實際無法收現
  • 企業在期末進行大額銷售，但客戶有實際的「返貨」協議

虛假資產：

  • 應收帳款中包含無法收回的壞帳，但沒有適當提列
  • 存貨中包含滯銷品或報廢品，但按成本入帳

隱藏負債：

  • 企業向關聯方借款但未在報表中明示
  • 企業進行表外融資，避免在負債表上呈現

郭榮芳強調，這些虛假操作的目的通常是美化報表以便進行融資或圈錢。一旦被揭露，企業的股價往往會大幅下跌，投資者的損失可能是災難性的。

11.6 風險評估的完整檢查清單

基於以上分析，郭榮芳總結了一份投資者或內部管理層可以用來進行風險評估的檢查清單：

財務指標檢查：

  • [ ] 應收帳款佔營收的比例是否合理？與同業相比是否偏高？
  • [ ] 存貨佔營收成本的比例是否合理？是否異常低？
  • [ ] 營業活動現金流是否為正？佔淨利潤的比例是否至少 60%？
  • [ ] 投資活動現金流是否能由營業現金流覆蓋？
  • [ ] 負債比例是否在合理範圍內（通常不超過 60%）？
  • [ ] 公司是否有持續發展的融資依賴（年年都要借新錢來支持運作）？

業務質量檢查：

  • [ ] 銷售客戶是否集中在少數客戶？（應該有多元化的客戶基礎）
  • [ ] 大客戶的規模、資本額與採購金額是否相當？
  • [ ] 企業的核心業務是否為利潤主要來源？（還是主要靠業外投資）

組織與治理檢查：

  • [ ] 公司的組織架構是否清楚且合理？
  • [ ] 是否存在不必要的複雜層級或關聯企業？
  • [ ] 股東與管理層的權責是否清晰？

趨勢檢查：

  • [ ] 各指標的三年或五年趨勢是什麼？是否在惡化？
  • [ ] 與同業的對標情況如何？
  • [ ] 企業是否在進行持續的、可解釋的策略調整，還是數據異常波動？

如果檢查中發現 3 個或以上的紅旗項，投資者應該提高警惕，深入調查或考慮迴避這個投資機會。

11.7 回到企業經營層面

從地雷股的識別回到企業經營層面，郭榮芳的建議是：

對於經營者：

不要為了短期的帳面業績而進行不可持續的操作（如虛假銷售、過度投資）。長期的健康經營——真實的營收、可控的成本、健康的現金流——才是企業長期價值的基礎。

對於投資者或債權人：

不要被表面的 EPS 或營收數字迷惑。深入分析現金流量、應收帳款、存貨和融資情況，才能識別出那些表面光鮮但內實脆弱的企業。

對於內部管理層或審計人員：

財務報表是企業的「體檢報告」。認真閱讀這份報告，不只看利潤，更要看現金流、看資產品質、看融資情況，這樣才能早期發現企業可能存在的問題，及時採取行動。

附錄一：核心公式速查表

損益表相關公式

`

營業毛利 = 營業收入 - 營業成本

營業毛利率 = 營業毛利 / 營業收入

營業利益 = 營業毛利 - 營業費用

(= 營業收入 - 營業成本 - 營業費用)

營業利益率 = 營業利益 / 營業收入

本期淨利 = 稅前淨利 - 所得稅費用

淨利率 = 本期淨利 / 營業收入

EBITDA = 本期淨利 + 利息 + 稅金 + 折舊 + 攤銷

`

資產負債表相關公式

`

資產 = 負債 + 權益

負債比率 = 負債 / 資產

權益比率 = 權益 / 資產

流動比率 = 流動資產 / 流動負債

`

資產周轉與報酬率公式

`

資產周轉率 = 營業收入 / 平均總資產

資產報酬率（ROA）= 本期淨利 / 平均總資產

= 淨利率 × 資產周轉率

= (本期淨利 / 營業收入) × (營業收入 / 平均總資產)

`

營運管理指標

`

應收帳款收現天數 = 應收帳款 / 日均銷售額

存貨平均售貨天數 = 存貨 / 日均銷售成本

應付帳款支付天數 = 應付帳款 / 日均購置成本

營業周期 = 應收帳款收現天數 + 存貨售貨天數 - 應付帳款支付天數

`

現金流量相關公式

`

自由現金流量 = 營業活動現金流量 - 資本支出 - 稅金

或者

自由現金流量 = 營業活動現金流量 - 營業活動所需的資本支出

`

附錄二：不同企業類型的典型特徵

高利潤率、低周轉率的企業（如台積電）

財務特徵：

  • 毛利率：40% 以上
  • 營業利益率：30% 以上
  • 淨利率：20% 以上
  • 資產周轉率：0.5 倍以下
  • 資產報酬率：仍然較高（因為利潤率極高）

競爭優勢基礎：

  • 技術領先、品牌溢價、獨特資源或市場地位

經營重點：

  • 研發投資（維持技術優勢）
  • 品牌建設（維持品牌溢價）
  • 利潤率維護（避免價格戰）

風險點：

  • 如果競爭對手追上技術，利潤率會迅速下降
  • 如果市場需求變化，高成本結構的企業調整困難

低利潤率、高周轉率的企業（如鴻海、八方雲集）

財務特徵：

  • 毛利率：10% 以下
  • 營業利益率：2-5%
  • 淨利率：2-3%
  • 資產周轉率：1.5 倍以上
  • 資產報酬率：通過高周轉彌補低利潤率

競爭優勢基礎：

  • 規模經濟、成本優勢、高效運營

經營重點：

  • 成本控制（毫厘之差就會影響利潤）
  • 流程優化（提高周轉速度）
  • 市場擴張（做大規模以分攤固定成本）

風險點：

  • 成本控制失誤會直接侵蝕利潤（利潤率本來就低）
  • 市場飽和時無法再提高周轉率
  • 容易陷入價格戰

結語：財報力的本質

郭榮芳課程的核心傳達，不是教你變成財務分析師，而是教你：

1. 看懂企業的真實面貌

不是被表面的增長、高 EPS 迷惑，而是能夠透過財務報表看出企業真實的獲利能力、現金狀況、風險水準。

2. 做出更聰明的決策

無論你是經營企業、管理部門、還是進行投資，有財報力就能做出基於事實而非猜測的決策。

3. 識別危機

在問題還沒爆炸時，就從報表中看出端倪，及早採取行動。很多企業的倒閉都不是突然，而是有跡可循的。

4. 理解商業的本質

企業為什麼賺錢，是因為什麼模式，需要什麼投資來維持，這些問題的答案都在財務報表裡。

財報力就是這樣一種能力：用數字去理解世界，用邏輯去判斷未來，用事實去做決策。

郭榮芳多次強調：「沒有數量化就不能管理。」反過來說，有了數量化（財報），你就掌握了管理和決策的基礎。
